Project Overview: This project implements key generation, public key encryption, and decryption for a standard ascii text file.

included files: Makefile, public-key-cryptosystem.c, README.txt

To compile : make

Usage :

./public-key-cryptosystem keygen 

./public-key-cryptosystem  key.txt plaintext.txt -e (for encryption)

./public-key-cryptosystem  key.txt ciphertext.txt -d (for decryption)
